# API de Documentos

## API CRUD para documentos em um banco MONGODB

## Para acesso a API deve autenticar pela api de autenticação.

## Tecnologias utilizadas: Java com Spring Framework.

### Exemplo de uma chamada

### POST http://localhost:PORT/api-documentos/documentos

### HEADER Authorization : "tokenJWT"

### BODY da chamada { [ "texto : "" " ] }
